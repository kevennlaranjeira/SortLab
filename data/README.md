# Dados de exemplo

Os arquivos desta pasta mostram o formato usado pelo SortLab:

- a primeira linha indica o tamanho do vetor;
- as linhas seguintes contêm um inteiro por linha;
- as saídas mantêm o mesmo formato, mas com os valores ordenados.

Durante a execução, o programa gera novas entradas, saídas e arquivos de tempo dentro da pasta do algoritmo escolhido.
